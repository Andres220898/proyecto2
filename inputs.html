Ejercicio de inputs


Descripcion practica de la etiqueta input en html.
Tipo de INPUT Botón INPUT Función
Text Permite al usuario ingresar texto alfanumérico.
Email Adecuado para introducir direcciones de correo electrónico, incluyendo la tecla @
Password Similar al tipo de entrada de texto, pero oculta los caracteres ingresados, útil para contraseñas.
Number Permite al usuario ingresar números, con opciones para controlar el rango y los incrementos.
Url Diseñado para ingresar URLs (direcciones web), con verificación automática del formato.
Date Permite al usuario seleccionar una fecha de un calendario interactivo.
Time Permite al usuario seleccionar una hora específica.
Checkbox Proporciona una casilla de verificación que puede estar marcada o desmarcada.
Radio A B C Permite al usuario seleccionar una opción de un grupo de opciones exclusivas.
File Permite al usuario cargar archivos desde su dispositivo.
Submit Crea un botón que envía un formulario cuando se hace clic, generalmente utilizado para enviar datos.
Reset Crea un botón que restablece los valores del formulario a sus valores originales.
Buttom Crea un botón que puede tener una función personalizada mediante JavaScript.
Color Permite al usuario seleccionar un color mediante un selector de colores.
Range Crea un control deslizante que permite al usuario seleccionar un valor dentro de un rango definido.
Hidden Un campo oculto que almacena datos en el formulario pero no se muestra al usuario.
Search Utilizado para entradas de búsqueda, puede tener estilos específicos en algunos navegadores.
Tel Permite al usuario ingresar un número de teléfono. Puede tener características específicas de formato para números de teléfono.
Month Permite al usuario seleccionar un mes y un año utilizando un selector de mes.
Week Permite al usuario seleccionar una semana y un año utilizando un selector de semana del calendario.
